### Install API

```bash
# Clone this repository
$ git clone https://github.com/cesaraugustomt/nlw-03-discovery-happy.git
# Go into the repository
$ cd backend

# Install dependencies
$ yarn install

# Run Migrates
$ yarn typeorm migration:run

# Start server
$ yarn dev

# running on port 3333
```

### Install Front-end

```bash
# Clone this repository
$ git clone https://github.com/cesaraugustomt/nlw-03-discovery-happy.git

# Go into the repository
$ cd web

# Install dependencies

# Run
$ yarn start

# running on port 3333

```
